## Install

Clone the repo using

```
$ git clone https://github.com/HarrisonHenri/ignite-react-native-stream-data
```

```
$ cd ignite-react-native-stream-data
```

Install all the required dependencies

```
$ yarn
```

Then execute

```
$ expo start
```
